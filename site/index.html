Start here

One command. A complete local inventory.

The audit reads supported skill folders and local session evidence. Your files stay on your machine.

Terminal 
$ npx github:Ryan-yang125/skill-manager audit
Install the Agent Skill 
$ npx skills add Ryan-yang125/skill-manager --skill audit-agent-skills -g -y
READ

Inventory the roots

Scan shared, Codex, and Claude global skill folders. Capture each path, package source, file size, and instruction content.

TRACE

Match local evidence

Search Codex and Claude session logs for traceable references. Every count points back to evidence on disk.

DECIDE

Keep, review, archive

Use age, overlap, context size, and evidence coverage to make a human decision. Archive and restore stay reversible.

The inventory layer

Know what loads before the prompt.

Agent skills multiply across folders, plugins, and package sources. Skill Manager turns that filesystem sprawl into an inspectable library.

~/

Origin and path

See the exact folder, agent root, package metadata, source URL, and every detected location for a skill.

EV

Usage evidence

Review usage count, last reference time, session kind, confidence, and the local evidence path behind each match.

CTX

Catalog context estimate

Estimate the discovery metadata formed by each skill name and description, then compare that catalog footprint across the inventory.

↺

Reversible cleanup

Write a durable ledger before moving a skill. Restore returns it to the original path and protects existing folders.

Evidence over guesses

Every signal stays inspectable.

Skill Manager records the session path, evidence kind, timestamp, and confidence behind every usage match. Cleanup reports export as Markdown and JSON for review.

Interpretation rule: zero matched evidence means “review this skill.” Session coverage can be incomplete, and rare skills may still be essential.
