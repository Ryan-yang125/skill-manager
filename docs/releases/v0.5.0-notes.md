# Skill Manager 0.5.0

Skill Manager 0.5.0 ships the new Electron desktop app for managing local agent skills.

## Highlights

- Browse installed skills across `~/.agents/skills`, `~/.codex/skills`, and `~/.claude/skills`.
- Read rendered `SKILL.md` content with frontmatter hidden from the reading view.
- Inspect local package/source metadata, locations, usage count, last used time, and evidence.
- Search and sort by latest added or usage count.
- Archive skills with a durable local ledger and restore them to their original path.
- Export cleanup reports as Markdown and JSON.
- Use Mosaic light and dark themes with English and Chinese UI.

## Privacy

Skill Manager runs locally for inventory, usage analysis, archive, restore, and report export. The first public release sends no telemetry.

## Install

Download the artifact for your platform and verify `SHA256SUMS.txt` before launching.

macOS direct-download builds use ad-hoc signing. If Gatekeeper blocks first launch, right-click the app and choose Open.

Windows direct-download builds may show SmartScreen until code signing is configured. Verify the checksum before launching.
