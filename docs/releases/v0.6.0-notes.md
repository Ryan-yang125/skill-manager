# Skill Manager 0.6.0

Skill Manager 0.6.0 adds a low-friction CLI, an installable Agent Skill, and a public documentation site to the existing local-first desktop app.

## Highlights

- Run an evidence-backed Agent Skills audit from the terminal.
- Export stable JSON or Markdown reports for agents and automation.
- Inspect each detected skill, its source, path, context estimate, and local usage evidence.
- Preview archive operations, archive with explicit confirmation, and restore through the durable ledger.
- Verify stable content hashes before restore and stop on archived-content changes.
- Install the `audit-agent-skills` Skill and let an agent explain the audit before any filesystem change.
- Read focused guides for Agent Skill locations, unused-skill evidence, duplicate paths, context estimates, and safe recovery.
- Verify published files with SHA-256 checksums and GitHub build-provenance attestations.

## Privacy and safety

Audits run locally and send no telemetry. Missing evidence is reported conservatively. Archive and restore remain explicit, recoverable actions with path conflict checks.

## Install

Run the CLI directly from GitHub:

```bash
npx github:Ryan-yang125/skill-manager audit
```

Install the CLI with Homebrew:

```bash
brew install Ryan-yang125/skill-manager/skill-manager
```

Install the Agent Skill:

```bash
npx skills add Ryan-yang125/skill-manager --skill audit-agent-skills -g -y
```

Desktop packages for macOS, Windows, and Linux are attached to this release together with checksums and the standalone CLI package.
