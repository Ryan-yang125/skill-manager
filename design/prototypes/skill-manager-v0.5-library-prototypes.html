# Agent Browser

Use this skill when browser state matters.

## Quick Start

Run browser actions through the local CLI:

  npx agent-browser open https://example.com
  npx agent-browser click "Sign in"
  npx agent-browser screenshot

Prefer durable selectors and verify the resulting page state before handing work back.
