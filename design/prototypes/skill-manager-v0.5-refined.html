# Agent Browser

Use this skill when browser state matters and the task needs a repeatable action trail.

## Quick start

npx agent-browser open https://example.com
npx agent-browser click "Sign in"
npx agent-browser screenshot

Prefer stable selectors. Verify the resulting page state before returning work.
