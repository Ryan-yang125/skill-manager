# Community Code of Conduct

Skill Manager welcomes people who want to improve local Agent Skill tooling, documentation, safety, and compatibility.

## Expected behavior

- Be respectful, specific, and constructive.
- Discuss technical evidence and reproducible behavior.
- Protect personal paths, session content, credentials, and private skill material in fixtures and screenshots.
- Give contributors credit and accept good-faith correction.

## Unacceptable behavior

- Harassment, threats, discrimination, or personal attacks.
- Publishing another person's private information.
- Submitting malicious skills, prompt injection, unsafe scripts, or deceptive fixtures.
- Repeated disruption of project discussions or release work.

## Enforcement

Maintainers may edit or remove contributions and restrict participation when needed to protect the community or users. Report sensitive conduct or security concerns through the repository's private security advisory flow.

This policy applies in the repository, Discussions, issues, pull requests, and project-managed community spaces.
